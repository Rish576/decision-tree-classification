# Decision Tree Classification

This project demonstrates a machine learning classification model using the Decision Tree algorithm.

The notebook includes:
- Data loading and preprocessing
- Training a Decision Tree classifier
- Model evaluation and accuracy calculation
- Prediction on new data

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab

## How to Run
1. Open the `.ipynb` file in Google Colab or Jupyter Notebook.
2. Run all the cells in order.
3. Modify the dataset path if required.
